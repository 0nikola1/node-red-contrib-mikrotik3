A node that work with Mikrotik WiFi routers.

Device: Setup Mikrotik settings like host, port (8278 - default), username and pass.

Action:
  • Log: returns log as msg.payload. Mikrotik command: /log print
  • Connected WiFi: returns list of registered WiFi devices as msg.payload. Mikrotik command: /interface wireless registration-table print
  • Reboot: reboots Mikrotik. Mikrotik command: /system reboot
  • Raw: uses msg.payload as command. To pass command with parametrs use JSON object with "command" property as msg.payload. For example: {"command":["/system/script/run","=.id=myscript"]}. This example runs script myscript

In case of success msg.success is set to true.
